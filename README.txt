Cachaceiro Viajante
Pré-requisitos
Para executar o programa, é necessário ter o Python instalado na máquina. Recomenda-se o uso de uma versão do Python 3.
Além disso, certifique-se de ter as bibliotecas necessárias instaladas. Você pode instalá-las utilizando o seguinte comando:

pip install pulp networkx matplotlib pandas
Como usar
    1. Baixe o código-fonte do programa ou copie-o para o seu ambiente de desenvolvimento.
    2. Abra um terminal ou prompt de comando e navegue até o diretório onde o programa está localizado.
    3. Execute o programa com o seguinte comando:
python CachaceiroViajante.py

    4. O programa criará automaticamente um arquivo Excel chamado cachaceiro_data.xlsx com os dados dos bares e suas respectivas distâncias e níveis de intoxicação.
    5. O resultado da execução, incluindo a rota encontrada, a distância percorrida, a intoxicação por bar e a intoxicação total, será exibido no terminal.
